今回お話しするのは、みんなが待ちに待った（そんなことはない）[P5.js](https://p5js.org/) です。名前の通り、`.js` で終わるものは JavaScript の package で、これはアート作品の制作に使えます。もしかすると [D3.js](https://d3js.org/) のような package を聞いたことがあるかもしれません。両者の違いは：D3.js はデータ志向、P5.js はアート志向です。どちらも可視化に分類されますが、実はかなり違います。例えば、D3 のデータはたいてい意味があるものですが、P5 のデータはほぼランダムな乱数です。初心者には、こう捉えるほうが分かりやすいかもしれません（？）。

実用性で言えば、筆者は絶対に D3.js をおすすめします。面接でプラスになることもあります。チャートや円グラフなどたくさんの図表を作れるからです。一方 P5.js は本当にアート作品を作るため（修身養性？）に学ぶものです。ちょっと覗いてみたいという方は、この記事を読んでみてもいいと思います。この記事は P5.js が何をするものかを少し紹介するものです。興味がなければ無理しなくて大丈夫。それでも興味があるという皆さん、始めましょう！

## Init

まずお話しするのはキャンバスです。一枚の絵で最も大切なのはキャンバスです。キャンバスがあれば、アーティストはその上で存分に創造力を発揮できます。

でもキャンバスを生成する前に、まず [OpenProcessing](https://openprocessing.org/) のアカウントを登録する必要があります。これは Open source のプラットフォームで、無料で登録できます。登録したら私たちの旅を続けられます〜

![openProcessing](https://i.imgur.com/YJI6iF7.png)

## Create

次はキャンバスの上で創造力を発揮しましょう！

何も変えずに「三角の実行ボタン」を押してみましょう。

![OpenProcessing](https://i.imgur.com/MVfRLzn.png)

> あなたは今や P5.js の絶対領域に入りました！

まず少し遊んでみて、P5.js のパワーを体験してみてください。ここまで来ると、最終成果まであとわずかです（えっ、まだ何もやってないって？！）。

焦らないで、コードはここに来ましたよ。
筆者が少し時間をかけてコメントを付けました（~~翻訳。英語は本当に、本当に難しい~~）。

```javascript
// 只會在初始化的時候執行一次
function setup() {
  // 初始化畫布
  createCanvas(windowWidth, windowHeight)
  // 背景：只有一個數值 (0 ~ 255) 黑 ~ 白
  background(200)
}

// 每個 "frame" 都執行
function draw() {
  // 當滑鼠按下的時候才畫
  if (mouseIsPressed) {
    // 要畫的顏色：(0, 0, 0) ~ (255, 255, 255) => 黑色 ~ 白色
    fill(0, 0, 0)
    // 畫線，上一個 previous (pmouseX, pmouseY) 到現在滑鼠位置(mouseX, mouseY)
    line(mouseX, mouseY, pmouseX, pmouseY)
  }
}
```

そうすれば、お絵かきソフトのように絵を描けます。機能がたくさんあります。例えば `line` という[関数の使い方](https://p5js.org/reference/#/p5/line)は、毎回ドキュメントを調べないと使えません。それくらい種類が豊富だからです。

[P5.js のドキュメント](https://p5js.org/reference/)も参照してください。

点、線、四角形など、描きたいものはだいたいドキュメントに載っています。

残りはあなたの超次元の力次第！

## Summary

今日は、P5.js を使えばわずか数行のコードで素晴らしい作品が作れることを学びました。

簡単なお絵かきソフトを少し一緒に体験しました。大きなプロジェクトではありませんが、考えてみてください：「P5.js を使わず、ネイティブの HTML/CSS/JavaScript だけでお絵かきソフトを書くとしたら、何行のコードが必要でしょうか？」でも P5.js は **面倒な重労働** を全部包んでくれているので、アーティストは創作に集中できます。もちろん、これは P5.js のほんの一部です。もし興味があれば、もっと **ジェネラティブアート** にも挑戦してみてください。P5.js の中にループを入れてみましょう！
