## 启蒙

说到嘸蝦米，大家脑中一定都是以前国、高中时，电脑课才会教的东西（暴露年龄了）。会用的应该都是老人了吧。但笔者才学了 2 年左右，目前已经可以流畅地聊天、打文章了。当然有时候还是有打不快的字，但对我来说已经是利大于弊了。

最一开始是看了 Papaya 老师的介绍影片，看到老师嘸蝦米打得「快、狠、准」，立马被深深吸引。之后笔者又看了好几次影片，最后狠下心来学习一波。依现在的时间点算起来确实是 2 年前左右开始从零开始学习的。

> by @Papaya 老师的嘸蝦米介绍

在此之前，我完完全全是新手小白。以前只有在电脑课时不小心切换到嘸蝦米，胡乱打几个字，然后就无缘了。可能小时候没电脑，连一般的注音中打都打不快。想说反正本来中打就不快了，切换的成本也不高，就干脆切换过去顺便练一下好了。

嘸蝦米吸引我的地方，除了最优势的不用选字外，它可以只用「A ~ Z，共 **26 个字母**」就可以完成中打输入。你可能觉得这没什么，除了比注音少了一些按键（ㄅ、…、一声、…等等，**共 41 个**），更少的输入可以打更快。

> 试试输入以下：「1 生、 2 人、 3 餐、 4 季」

<center>
  <img src="https://hackmd.io/_uploads/B1HXrmRTn.gif" class="post-image-width" alt="嘸蝦米 demo" />
</center>

> 对，就是封面的图（Gif 是原速度没有加速喔）

要么你切换了数次「中/英」模式；要么你使用独立的数字键，但两者都比较费时。世界上最远的距离就是从键盘中间到方向键（独立字键又更远了）。但是 **嘸蝦米可以直接打出数字**，因为它没有占用任何数字键。数字也是嘸蝦米有趣的地方，晚点会再提到。试想其他的符号呢？如：`!@#+-()[]{}` 等等，如果读者刚好用到这些符号，在嘸蝦米中都不用切换啦，是不是很棒呢。

再来上面也提到，嘸蝦米只需要 26 个字母就可以，但是你要打得快，英打也要有一定的要求。如果你英打本来就很快那很好；如果不快那也很好，因为练好嘸蝦米也可以 **一定程度的提升你的英打水平**，可以一起练起来的概念。

所以基于以上总总原因，我决定要来入坑嘸蝦米（才不只是因为帅呢）。再来会聊聊我的学习过程，也许是最无趣的但也是最重要的，希望上述的内容有些点可以打到你。

## 苦练

笔者下定决心之后，每天都到官网的线上练习练个 10～30 分钟。一开始真的不知道在干嘛，一堆字根要记、要练英打。本来想说 A~Z 要打也就 26 个字母，应该也不会太难，但有的字根结合了形、音、义，就会觉得很难。但也可能是我那时候同时要忙很多事，2 年前我也差不多在全职转职中、学习 Vim、练英打、练嘸蝦米，所以学得比较慢。如果读者的时间比较多、比较没其他学习压力，一定是可以比笔者快很多的啦！

虽说叫记字根，但其实嘸蝦米的字根都有依据的，因此 **用上一丁点联想力，很多字根基本上不用背**。基本字根都熟悉后，基本上就可以稍微打打字了。

整体来说，我觉得官网的练习小网站还不错，慢慢地一个一个从好记的字根开始，再到需要一些联想力的字根。我差不多每天上个 10～30 分钟，差不多一个月后，可以打 50% 的字了。心态上不要觉得它很难，把它当作游戏。今天学了「A 技能」，一天一天学到了「Z 技能」，等到都学完了，就可以来个组合技降龙十八掌了（误）。

## 迁移

当你大概 90% 的字都会打的时候，并且速度大概 40 左右时，就表示到了可以直接上嘸蝦米打中打了。除非你是需要打游戏时，需要大量的嘴炮需要打非常快，不然一般使用情况大约 60 就很够用了。所以真的 **差不多 40** 的时候就可以直上了～

嘸蝦米官网也很贴心地准备了试用版。不过它的试用版呢～很佛心，**是可以一直免费用的**，只是每三个月要重新安装一次。笔者一开始直上也是先用试用版的，大概用了一年才换付费版的（这边也可以先不买，文章最后会再提到免费的替代方案，但笔者还是有支持正版的）。

当然，一路走来练了这么多字根、完整的字，再来就是实战了。实战也是最快提升速度的时候。笔者刚换过来的时候也很卡，一定卡的啦。但也好像每个人都是这样来的，没什么好怪的，就怪自己不够熟练，继续练习就对了！

笔者会将常常遇到不会打的字、不熟的字，先用笔记软体记下来（有要拿去练习的也欢迎），然后用「最简码」练习一次，隔天再练习一次。久而久之这些词语就一个一个就熟了，会慢慢变成反射动作。

## 一时打字一时爽，一直打字一直爽

当你字都差不多会打的时候，你以为嘸蝦米就这样了？你错了！**好玩的才正要开始呢！** 如果你前面的某个阶段还不熟，就先看到了这一段的话，再撑一下！到这个阶段时，嘸蝦米的实力才会发挥出来，但前提是你要撑到这一阶段，可以再去把字根练熟一点。

在嘸蝦米的世界中，有所谓的「一字根表」，也就是 A~Z。如字面上的意思，打 **一个英文字 + 空格** 就可以打出一个中文字了。

例如：`A => 對`、`D => 的` ……等等。像这种字都是可以神速打出来的。

这些通常都是精挑细选的常用字或是不好打的常用字。当中还有一个特别的「中文数字」，在嘸蝦米的设计上，也通通都是一字搞定。因此很推荐给常常需要打中文字数字的读者，如：公文、笔录等等。**中文数字在注音很多字都是撞同音**，几乎 ~~一定~~ 要选字。

```javascript
;(E) => 一
;(R) => 二
;(S) => 三
;(F) => 四
;(W) => 五
;(L) => 六
;(C) => 七
;(B) => 八
;(K) => 九
;(J) => 十
```

> 简单念念看看再对照英文，你又会发现新大陆了！

俗话说得好：「有一就有二」，当然嘸蝦米的世界中，也有所谓的「二字根表」，也就是 AA、AB、…、ZZ。如字面上的意思，**两个英文字 + 空格** 就可以打出一个中文字了。也就是说二字根表，共有 26 × 26 个字，通常都是常用字的简码。有趣的是一个字可以有好几种打法，一般的、简码的，但是都不太用选字。

所以嘸蝦米有这些东西有什么用？这背后的意思是：「假设熟练程度一样，跟注音比起来，嘸蝦米的输入少很多就可以打出一个字。读者觉得哪个输入法打得快（而且又准）？」

除了种种嘸蝦米的优势：不用选字、只使用英文字母……等，但是从这里开始就是 **注音追不上的地方了**。只打一两个字母就完成一个中文字的输入，是注音做不到的（是可以啦，但就只是注音文，你懂的）。所以继续好好练你的嘸蝦米。现在打不快也没关系，但至少你打得省力又不用选字。先速度练到 50～60，先这样就很够了。哪天突然想加强速度了，再回来练也行。你会发现以前注音练得要死，总是卡在一个瓶颈，嘸蝦米轻松就突破，甚至超越了！

## 番外篇：rime-liur

`rime-liur` 是基于 `RIME` 输入法设计的全功能嘸蝦米方案。`RIME` 又称「鼠须管」，如果成为小蝦米的话，应该多少会听过。

> GitHub Repo：[hsuanyi-chou/rime-liur: 基于 RIME 输入法设计的全功能嘸蝦米方案](https://github.com/hsuanyi-chou/rime-liur)

`rime-liur` 是 @hsuanyi-chou 大大基于 `RIME` 开发，并且是 **开源出来的**（笔者已给 Star 支持一下）。对于我们一般使用者来说可以 ~~算是免费使用~~。那么 `rime-liur` 有什么特别之处呢？请见 GitHub 上的解说（逃）。

好啦，这边笔者挑几个好用的功能说明一下：

1. 可以直接用 `shift` 切换中/英，一般是只能使用 `ctrl + space`
2. 注音模式、拼音模式、字码反查
3. 可以客制化输入法的字体、主题、候选字颜色等等

之前刚好有朋友遇到不会使用「加词加字」功能（这算是官方就有的功能了，所以上面没有特别提到）。确实这对一般非开发者的人员来说，去改设定档是有些困难的。可能少了一个空白设定档就坏了（这是真的）。所以笔者想想就简单录了一个教学给朋友，对我来说也算是一个纪录。有需要的读者也可以参考看看囉。

<center>

[![嘸蝦米輸入法](https://img.youtube.com/vi/I9tlZFB2lvI/0.jpg)](https://www.youtube.com/watch?v=I9tlZFB2lvI)

</center>

> Rime 嘸蝦米的加词加字功能教学, by @Benben

例如：影片中一开始的预设是 `name + 空格` 就会打出 `我的名字`。我在后面加了 `Benben`，所以之后只要打 `name + 空格` 就会打出 `我的名字：Benben`。加名字、信箱、地址等等只是基本。后来我再影片中加入了 `hw; + 空格` 就会打出 `Hello World`，其中取了 h 作为 hello、w 作为 world，应该很好理解。

眼尖的读者就会发现：我在 `hw` 后加入了一个 `;`，这是刻意的。因为在嘸蝦米中 1～4 英文字母的组合，很大的几率会碰撞（俗称「被占用」）。所以笔者加了一个符号当作后缀。当然读者也可以自行发挥创意。所以我说 **好玩的才正要开始呢**！

## 小结

嘸蝦米当然不是看完一篇文章就可以简单学会的。大量的练习是必要，但付出的之后都会成为美丽的果实。省下的是你的时间。而俗话说得好「时间就是金钱」。假如在打字上每天帮你省下 10 分钟就好，一个月就是 5 个小时，拿去做更多的运用不是很好吗？

例如：假如你是客服或是卖家之类的，有一些常用的词，都是可以加入到嘸蝦米的「加词加字」功能了。设定完之后，随便打几个字就跟用喷的一样，其他同事看了一定吓吓叫，也是你的省时好帮手。诚心推荐给常需要输入大量文字的工作者。

嘸蝦米的水也很深、学习曲线高，但是之后的天花板也很高，可以玩的很多。也还有很多嘸蝦米的变体，例如：这篇文章中的 `rime-liur` 只是其中之一，还有很多好玩的功能、插件等着你去探索。

后来笔者请教一些前辈（大蝦米）们，真的很感谢前辈们愿意分享。希望这个「快失传的输入法」可以传承下去。这么好用的神器不能只有我知道啊！

以下整理一些嘸蝦米密技：

1. 「,,SP」快打模式（最简码）
2. 「,,C」简体模式
3. 「,,CT」打繁出简模式
4. 「,,J」日文模式
5. 「,,T」切回繁体模式
6. 「,,X」密语解码模式
7. 「、」同音字模式
8. 「、；」注音输入模式（一次性）
9. 「,,box」加字加词模式
10. 右键 > 内容，查字根或注音模式
