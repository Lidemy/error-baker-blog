## 前言

！**这篇文章算是偏心态跟自我提升的部分，也算是记录一个故事。**

时间过好快啊，不知不觉 2022 也过完一半了！年度目标还算有达成，下半年度要提升英文了。也算是讲很久了，所以 6 月底跟朋友们报了 [VoiceTube 挑战赛](https://hero.voicetube.com/challenge-youth)（9 折优惠码：v_rnc0xg）。究竟破英文的我会不会挑战成功呢？三个月后见。

我是 [Lidemy 第五期](https://bootcamp.lidemy.com/) 的学生（也可能是最后一期了？），转职前端也半年多了。转职的血泪史这边就不多做解释了。开心认识 Lidemy 的各路大神。

虽然已经是前端工程师了，也依然保持着持续学习的心态。对于前端领域的热情，我也不知道会烧到什么时候，但目前烧了 1～2 年，没意外的话应该还会继续。为了要成为资深前端，领导这个技能也算必点了。那么如果公司都是大神，没有人可以带怎么办？

只好自己去找囉（？）

结果，我很幸运地被别人找了，所以有了这篇文章。究竟是不是命运的安排呢？我也不知道。

找我的人是 K。我们是在 [Zero To Mastery (ZTM)](https://zerotomastery.io/community/developer-community-discord) 的社群上相遇的。ZTM 系列的课程也很棒，是 Udemy 上很火红的系列，这边也不做太多解释。基本上，社群上面几乎都是外国人，所以能遇到台湾人真的很幸运！

K 说他那边也找了 2～3 个还在学习的 junior，打算开读书会互相学习，非常欢迎有经验的人一起，所以我也就顺利地「混入」进去了，顺利地开启我的「微 mentor 之路」。

为什么笔者会想开始「微 mentor 计划」呢？让我们继续看下去～

## 星火燎原

所以我是谁不重要，重要的是，在很久以前、遥远的银河系……

(A long time ago in a galaxy far, far away....)

那时候的前端世界，还处于各种大战时期：浏览器大战、编辑器大战、前端框架大战……各种大战（~~当然还有星际大战~~）。战到老子学不动了，战到孩子都出生了，总之战得你不要不要的。

烽火连天、民不聊生，就在这时候，有一个人发起了「前端引路人计划」。

> 延伸阅读：[真正的 Mentorship program：前端引路人计划 | by Huli | Medium](https://hulitw.medium.com/mentorship-program-350db93d5c9c)

「前端引路人计划」是一个免费的活动，主要是由一个「mentor」带领其他工程师一起成长，不局限于前端，要做的 project 也不限制，还算自由。重点是免费！

这微小的星火，是这战乱前端世界的一小盏明灯。引路人计划的前期阶段也许成果不尽理想，但每次的一小步都是燎原的基石，也是最后演变为程序导师计划的前身。

从非常早期的只收 5 个人左右，到第五期有快 100 人报名。因为有了助教系统所以可以招收更多的学生。挑战成功人数也有 25 人左右（这边是指完课后 2 个月内，找到工程师工作的人，但其实更多，因为很多人后来还是有找到，只是晚了一点）。另外那些没有成功的人也是不用付学费的，因为这是一个保证班，保证转职，但保证学习多少就是修行在个人了。

## 惺惺相惜

后来顺利加入了 Lidemy 前端导师计划第五期，非常地开心，觉得自己大概是世界上最幸运的人了。

因为像老师这么好的人，这世界上已经很少了。出了社会才明白现实的残酷：前辈凭什么要教你技术？教会徒弟，饿死师父。

何况真正想学的人也是少之又少。有一部分是走马看花的伸手党也不在少数。但参加程序导师计划的都是有初步筛选的人。我看你骨骼精奇，是百年难得一见的练武奇才，要不要来当 ~~爆肝~~ 工程师？

呃……好啊！

总之能跟一群有目标、有理想的人一起奋斗，感觉真的很棒！

比较不一样的是：这个计划 **没有固定的 Final Project**。一般来说，很多 boot camp 都会有一个规划好的 Final Project，最常见的就是：电商购物车、Twitter、Netflix Clone 之类的。所以能找到一样有想法的人一起做 Final Project 本身就是一件难能可贵的事。再加上整个项目从零开始，从 user story 的规划、figma 的制作、前后端的实作，甚至到部署，全部只有你跟你的组员们一起。有问题可以询问助教或老师，但基本上都是要跟组员们独立完成。虽然辛苦，但最后完成真的很有成就感！

其实这边的惺惺相惜指的不只是老师，也还有组员们、同学们。渺小的我能找到这样的环境、同学、助教们，真的是很难能可贵。

每天写 code 写到半夜的日子，我大概一辈子不会忘记吧！

## 薪火相传

老师的始祖计划文中提到：

> 如果这一年过后你很感谢我这样免费带人，不用感谢我。你带着这份谢意也开一样的计划去帮助其他人，就是最好的回报了。

条件是：

> 一年之后你也要开启相同的计划，收的人数不能少于 2 个。

虽然我不是很资深，但也想着要回报老师。但谁知道想着想着，它就发生了，就是这么神奇。

接下来，就是换我薪火相传的部分了。来稍微介绍一下这次「微 mentor」计划吧。我们主要是使用 discord 作为交流的工具，但分享的时候使用 google meet，比较不会卡顿。

目前成员有（这边用简写代称）：

- K，主揪，强有力的自学者。目前在外商半导体产业，是写 python 处理数据的。基本上他已经看到 react 了，看得差不多了，然后再刷一些 leetcode 了。我觉得根本就可以去转职了 XD。

- S，目前还是学生，未来想往区块链发展。在学 solidity 中，也用 next.js 写了前台。但 react 的掌握度还算不高（？她自己说的）。但她主要也不是往前端发展，也可能往研究所前进。

- A，目前在资策会上课，正在学习 PHP 后台中。也是估计今年完课后，年底去求职。也有在看一些 PM 的课程，未来可能会往 PM 走也不一定。

- 我，废物前端新手，看到大家都这么认真，我真的是汗颜 QQ。

好啦，也许我没这么废。但还是得学着韬光养晦，把自己的光芒收起来，把时间、空间留给这些明日之星发挥吧！没啦，我就想当个废物。

那么就未来半年见囉，各位软件工程师们。

就让这渺小的星（薪）火烧下去吧。
