## Init

The first thing to talk about is the canvas. The most important part of any picture is the canvas — once you have one, the artist can let their creativity run wild on it.

But before generating the canvas, you need to register an [OpenProcessing](https://openprocessing.org/) account. It's an open source platform — free to register — and then you can continue our journey ~

![openProcessing](https://i.imgur.com/YJI6iF7.png)

> The [OpenProcessing](https://openprocessing.org/) site is just _cool_ (if a bit slow?)

After opening a new project, your screen should look like mine. The UX is pretty friendly (compared to other Adxxx products), but I'll walk through it briefly:

1. The triangle button right in the middle of the page — click it and your code starts running.
2. `</>` — no need to explain XD — that's how you get back to the code.
3. The Save button in the top right is for saving (professional Chinese-to-English translation?).
4. The right side has some settings; you basically don't need to touch them.

![OpenProcessing](https://i.imgur.com/97LoeAN.png)

> A very friendly interface — small but mighty.

First, you don't need to do anything — just open a project. What, you can start drawing already? Yep! That's right, it's just that easy!

## Create

Now unleash your creativity on the canvas!

Without doing anything, just press "the triangle run button."

![OpenProcessing](https://i.imgur.com/MVfRLzn.png)

> You've now entered the absolute domain of P5.js!

You can play around a bit first to experience P5.js's power. We're already close to our final result (wait, you say you haven't done anything yet?!)

Don't rush — here comes the code!
I spent some time adding comments (~~translation; English is genuinely so, so hard~~).

```javascript
// 只會在初始化的時候執行一次
function setup() {
  // 初始化畫布
  createCanvas(windowWidth, windowHeight)
  // 背景：只有一個數值 (0 ~ 255) 黑 ~ 白
  background(200)
}

// 每個 "frame" 都執行
function draw() {
  // 當滑鼠按下的時候才畫
  if (mouseIsPressed) {
    // 要畫的顏色：(0, 0, 0) ~ (255, 255, 255) => 黑色 ~ 白色
    fill(0, 0, 0)
    // 畫線，上一個 previous (pmouseX, pmouseY) 到現在滑鼠位置(mouseX, mouseY)
    line(mouseX, mouseY, pmouseX, pmouseY)
  }
}
```

Then you can draw on it like a little paint program. You'll see lots of features — for example, the [usage of the `line` function](https://p5js.org/reference/#/p5/line) — every time you'll have to check the docs to use them, because there's just so much.

You can also consult the [P5.js docs](https://p5js.org/reference/).

Drawing dots, lines, squares — whatever you want to draw, the docs basically cover it.

The rest is up to your superpower!

## Summary

Today we learned that P5.js lets us complete great creations with just a few lines of code.

I walked you through a simple little paint program. It's not a huge project, but consider: "Without P5.js — using only vanilla HTML/CSS/JavaScript — how many lines of code would it take to build a paint program?" Here P5.js has wrapped up all the **heavy lifting** for us, letting artists focus on creating.
